# pomodoro
 Pomodoro es un método de gestión del tiempo que divide el trabajo en intervalos de 25 minutos de concentración llamados pomodoros, seguidos de breves descansos de 5 minutos. Tras cuatro ciclos, se toma una pausa más larga de 15–30 minutos. Su objetivo es mejorar la productividad y mantener la mente fresca al evitar la fatiga acumulada.
